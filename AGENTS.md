# Agent instructions

Canonical instructions for AI coding agents working in this repo live in
[`CLAUDE.md`](CLAUDE.md) — read it before making changes. It covers code
structure (thin `MainActivity`, one concern per file), validation
(`./gradlew check` is the gate), and the pull-request rules: PR bodies must
follow `.github/PULL_REQUEST_TEMPLATE.md`, ticking only the checkboxes for
things that actually happened.
