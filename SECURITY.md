# Security Policy

While this is primarily a client-side visualizer app that only communicates on your local network (e.g., for smart lighting integrations), keeping our users' devices and home networks safe is still important to us.

## Reporting a Vulnerability

If you discover a security issue, please **do not open a public issue**.

Instead, please report it privately so we can fix it before the details become public:

* Use GitHub's private reporting feature: Go to the **Security** tab in this repository, select **Advisories**, and click **Report a vulnerability**.
* Or, reach out to the maintainers directly.

We will do our best to address the issue in the next update. Thank you for helping keep the project safe!
